# Landing Law Firm

Лендинг страница для юридической фирмы, созданная с использованием React, Vite и Tailwind CSS.

## Особенности

- Адаптивный дизайн
- Формы консультации
- Презентация команды
- Современные UI компоненты
- Оптимизированная производительность

## Технологии

- React 18
- TypeScript
- Vite
- Tailwind CSS
- ESLint

## Установка

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```
